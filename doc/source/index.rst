MAJIQ: Modeling Alternative Junction Inclusion Quantification
=============================================================

**MAJIQ** uses the results of RNA-seq experiments to analyze alternative
splicing.

*NOTE:* This documentation is for the new release ("Majiq V3"), if you are looking for v2 documentation, it can be found here: https://biociphers.bitbucket.io/majiq-docs-v2/
